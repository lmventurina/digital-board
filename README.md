School Digital Bulletin Board

A real-time, responsive digital signage solution for schools, built with HTML, Tailwind CSS, and Firebase.

Setup Instructions

Download the files:

index.html

styles.css

app.js

Upload to GitHub:

Go to your GitHub repository.

Upload all three files to the main (root) folder.

Enable GitHub Pages:

Go to your repository Settings.

Click on Pages in the left sidebar.

Under "Build and deployment" > "Source", select Deploy from a branch.

Select main branch and /(root) folder.

Click Save.

Configure Firebase (Important):

Go to your Firebase Console.

Navigate to Authentication > Settings > Authorized Domains.

Click Add domain and enter your GitHub Pages URL (e.g., yourusername.github.io).

If you skip this step, the board will stay blank or show an auth error.

Updating Content

Open the website on any computer.

Click the Gear Icon in the bottom right corner.

Make your changes (Announcements, Ticker, etc.) and hit Save.

The changes will reflect instantly on all screens displaying the board.
